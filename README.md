# wren-cli-hello-world

## Description
A POC for wren scripting language.

## Tech stack
- clang
- make

## Docker stack
- alpine

## To run
`sudo ./install.sh -u`

## To stop (optional)
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
